HTML to Figma by dMaya

Paste or upload the JSON payload generated by the dMaya converter. Do not drag the JSON file onto the canvas.

Try the full design tool at dmaya.ai

This plugin creates real Figma frames, rectangles, text, vectors, and image-fill rectangles from the payload. Temporary remote assets are downloaded during import and skipped if they have expired.

How to use
  1. Open dmaya.ai/html-to-figma.
  2. Paste a URL, HTML file, or AI-generated HTML and run the converter.
  3. Download or copy the generated JSON payload.
  4. Paste or upload that JSON here, then click Import into current page.
