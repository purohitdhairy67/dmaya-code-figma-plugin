HTML to Figma by dMaya

Paste or upload the JSON payload generated by the dMaya converter. Do not drag the JSON file onto the canvas.

This plugin creates real Figma frames, rectangles, text, vectors, and image-fill rectangles from the payload. Missing remote assets are skipped unless the converter embedded them as data URLs.
