## Short Description

Convert HTML, public URLs, Claude Code, Lovable, and AI-generated frontends into editable Figma layers.

## Description

HTML to Figma by dMaya connects the free dMaya web converter with Figma. Paste HTML, upload an `.html` file, or convert a public URL on dmaya.ai, then paste the generated payload into this plugin to create editable Figma frames, text, vectors, image fills, borders, shadows, and raster fallbacks where needed.

Use it for HTML to Figma, URL to Figma, Claude Code to Figma, Lovable to Figma, Cursor to Figma, Bolt to Figma, v0 to Figma, Replit to Figma, and other AI output to Figma workflows.

## Start Here

Create your import payload at https://dmaya.ai/html-to-figma

This plugin is the Figma importer. The web page is where you convert a URL, HTML file, or AI-generated HTML into the JSON payload this plugin can import.

## How To Use

1. Open https://dmaya.ai/html-to-figma in your browser.
2. Paste HTML, drop an `.html` file, or enter a public URL.
3. Click Convert.
4. Click "Copy for Figma" or "Download payload".
5. In Figma, run HTML to Figma by dMaya.
6. Paste or upload the payload inside the plugin.
7. Edit the imported layers freely.

The heavy browser rendering happens on dMaya's backend. For large pages, temporary image assets may be downloaded from dMaya's production R2 asset domain during import so the copied payload stays fast. Assets stored in dMaya asset storage are deleted shortly after conversion; import the payload into Figma before those links expire, otherwise you will need to reconvert the website to generate a fresh JSON payload.

## Network Access Disclosure

This plugin only requests network access to:

- `https://dmaya-prod-r2.dmaya.ai`

Why: dMaya's web converter may store large generated image assets and full-page snapshot tiles as temporary files instead of embedding them as base64 in the JSON payload. During import, the plugin downloads those temporary image bytes and creates Figma image fills.

The plugin does not require authentication, does not read unrelated Figma files, and does not upload existing Figma file content or user designs to dMaya. If a temporary asset URL has expired, that asset is skipped and the user can reconvert the website to generate a fresh JSON payload.

## Notes For Review

- The plugin imports JSON generated by `dmaya.ai/html-to-figma`.
- JSON files cannot be dragged directly onto the Figma canvas; users paste or upload the payload inside the plugin UI.
- The plugin only has production network access to dMaya's temporary asset domain for importing generated image assets.
- Privacy policy URL: `https://dmaya.ai/privacy`.
